# Examples README

## Overview
This folder contains various example scripts that demonstrate the functionality of the LFO Synthesis Harmonic Analyzer. Each script is designed to provide insights into different aspects of the analysis and synthesis process.

## How to Use
To utilize the example scripts, follow these steps:
1. Clone the repository or download the scripts directly.
2. Navigate to this folder in your terminal or command prompt.
3. Run the desired script using the appropriate command for your programming environment.

## Available Example Scripts

- **example1.py**: Demonstrates basic harmonic analysis using predefined waveforms.
- **example2.py**: Shows how to implement LFO synthesis with user-defined parameters.
- **example3.py**: Provides an advanced example of combining multiple harmonic analyzers for complex waveforms.
- **example4.py**: Illustrates how to visualize the output of the harmonic analyzer.

## Notes
Ensure you have all the necessary dependencies installed before running these examples. Refer to the main project documentation for setup instructions.
