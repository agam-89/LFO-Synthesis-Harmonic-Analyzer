# Utility Scripts

This README file describes the utility scripts available in this repository and their purposes.

## Scripts

### batch_export.m

#### Purpose:
The `batch_export.m` script is designed to facilitate exporting data in bulk from the analysis processes conducted in this project.

#### How to Use:
1. Ensure that your data is prepared and ready for export.
2. Execute the script in your MATLAB environment.
3. The script will prompt you to select the data to be exported and specify the desired format.

### config.m

#### Purpose:
The `config.m` script is intended for configuration settings related to data analysis parameters. It allows users to set various options before executing the main analysis routines.

#### How to Use:
1. Open the `config.m` file in MATLAB.
2. Adjust the parameters according to your analysis needs.
3. Save your changes before running the analysis scripts that depend on this configuration.

Feel free to modify these scripts according to your requirements!
